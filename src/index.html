Data-Sars é um dos projetos do API (Aprendizagem pro Projetos Integrados) da FATEC, desenvolvida pela equipe Syntax Squad, da qual eu faço parte. O Data-Sars é uma aplicação web cujo objetivo é ampliar e facilitar o acesso à informações baseadas em dados do Data-Sus Tabnet e outros sites do governo a respeito da Covid longa: os Sintomas pós Covid-19, no Vale do Paraíba-SP.
